## Правила подбора цвета текста

### Для светлых фонов (белый, бежевый, светло-серый)
- Заголовки: тёмные насыщенные (#1a202c, #2d3748)
- Текст: тёмно-серый (#4a5568, #718096)
- Акценты: яркие цвета палитры

### Для тёмных фонов (синий, чёрный, тёмно-серый)
- Заголовки: белый или очень светлый (#ffffff, #f7fafc)
- Текст: светло-серый (#e2e8f0, #cbd5e0)
- Акценты: яркие контрастные (#f6e05e, #68d391)

### Для цветных фонов (градиенты, яркие)
- Использовать полупрозрачные подложки под текст
- `background: rgba(255,255,255,0.85)` для светлой подложки
- `background: rgba(0,0,0,0.7)` для тёмной подложки
