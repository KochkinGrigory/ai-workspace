# Photorealistic Portrait Preset

Пресет для создания фотореалистичных портретов высокого качества.

## Описание

Этот пресет создаёт профессиональные фотореалистичные портреты с естественным освещением и высокой детализацией.

## Template

```
{prompt}, professional portrait photography,
natural soft lighting, shallow depth of field,
realistic skin texture, high resolution, 4k quality,
sharp focus on eyes, photorealistic, DSLR camera,
bokeh background, professional color grading
```

## Использование

```bash
python3 generate.py --style-preset example-portrait "woman in business suit"
```

## Примеры

**Input:** `elderly man with gray beard`
**Output:** Professional portrait of elderly man with gray beard, natural soft lighting...

**Input:** `young artist in creative studio`
**Output:** Young artist in creative studio, professional portrait photography...

## Параметры

- **Стиль:** Photorealistic
- **Освещение:** Natural, soft
- **Качество:** 4k, high resolution
- **Фокус:** Sharp on eyes
- **Background:** Bokeh (размытый)

## Когда использовать

- Профессиональные портреты
- Headshots для бизнеса
- Персональные фотографии
- Портфолио
