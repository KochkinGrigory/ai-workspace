# Best Practices - Nano Banana Image Generation

Коллекция best practices для создания высококачественных изображений разных типов.

## Назначение

Эта директория содержит **руководства для Claude** по созданию промптов для разных типов контента.

**Когда использовать:**
- Пользователь просит создать изображение определенного типа
- Claude обращается к соответствующему best practice
- Применяет рекомендации при формировании промпта
- Генерирует изображение с оптимальными параметрами

## Структура

Каждый файл best practice содержит:
- **Ключевые принципы** для данного типа контента
- **Структуру промпта** с конкретными примерами
- **Технические параметры** (освещение, ракурсы, композиция)
- **Типичные ошибки** и как их избежать
- **Примеры промптов** разного уровня сложности

## Как Claude использует Best Practices

### Пример 1: Портрет

```
Пользователь: "Создай профессиональное фото для LinkedIn"

Claude:
1. Определяет тип: портретная фотография
2. Читает: photography-portraits.md
3. Применяет best practices:
   - Профессиональное освещение (студийный свет)
   - Ракурс: легкий наклон головы
   - Фон: нейтральный, размытый
   - Выражение: уверенное, дружелюбное
   - Технические детали: 85mm, f/2.8, фокус на глаза
4. Формирует промпт с этими параметрами
5. Генерирует изображение
```

### Пример 2: Продукт

```
Пользователь: "Нужно фото часов для каталога"

Claude:
1. Определяет тип: предметная съемка
2. Читает: photography-products.md
3. Применяет best practices:
   - Чистый белый фон
   - Равномерное освещение без теней
   - Ракурс: 3/4 для объема
   - Отражения и текстуры подчеркнуты
   - Резкость по всему объекту
4. Формирует промпт
5. Генерирует изображение
```

### Пример 3: Природа

```
Пользователь: "Красивый закат в горах"

Claude:
1. Определяет тип: природа/пейзаж
2. Читает: photography-nature.md
3. Применяет best practices:
   - Золотой час (warm tones)
   - Правило третей в композиции
   - Передний план для глубины
   - Широкоугольный объектив
   - Детализация неба и текстур
4. Формирует промпт
5. Генерирует изображение
```

## Отличие от Style Presets

### Best Practices (этот раздел)
- **Для Claude:** руководства по созданию промптов
- **Цель:** понять КАК лучше промптить для разного контента
- **Использование:** Claude читает и применяет при формировании промпта
- **Гибкость:** адаптируется под конкретный запрос пользователя

### Style Presets
- **Для пользователя:** готовые шаблоны для серий изображений
- **Цель:** создать много похожих изображений (например, для презентации)
- **Использование:** пользователь явно указывает `--style-preset название`
- **Консистентность:** одинаковый стиль для всех изображений в серии
