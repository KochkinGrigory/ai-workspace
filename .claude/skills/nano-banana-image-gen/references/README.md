# References - Техническая документация

Техническая документация и руководства для работы с Nano Banana Image Generation.

## Файлы

### `api-documentation.md`
Полная техническая документация Google Gemini API для генерации изображений:
- Параметры API
- Форматы запросов и ответов
- Коды ошибок и troubleshooting
- Ограничения и лимиты
- Примеры кода

**Для кого:** Разработчики, технические пользователи

### `prompting-guide.md`
Руководство по созданию эффективных промптов:
- Ключевые принципы промптинга
- Структура промпта
- Компоненты (subject, action, environment, style)
- Технические параметры (освещение, композиция)
- Общие рекомендации

**Для кого:** Claude и пользователи, изучающие основы промптинга

## Различия между документацией

### `references/` (этот раздел)
**Общая техническая документация:**
- Как работает API
- Как правильно промптить в целом
- Технические ограничения
- Универсальные принципы

### `best-practices/`
**Специализированные гайды по типам контента:**
- Конкретные рекомендации для портретов
- Конкретные рекомендации для продуктов
- Конкретные рекомендации для природы
- И т.д. - детальные инструкции под каждый тип

**Аналогия:**
- `references/prompting-guide.md` = учебник "Основы фотографии"
- `best-practices/photography-portraits.md` = специализированный курс "Портретная съемка"

## Использование

### Для Claude
1. **Начальное изучение:** читай `prompting-guide.md` для понимания базовых принципов
2. **При конкретной задаче:** обращайся к соответствующему файлу в `best-practices/`
3. **При ошибках API:** смотри `api-documentation.md`

### Для разработчиков
1. **Интеграция API:** см. `api-documentation.md`
2. **Оптимизация промптов:** см. `prompting-guide.md`
3. **Специфичные типы контента:** см. `../best-practices/`
