## Использование

**Примеры запросов:**

```
"Создай презентацию для Оптика Будущего"
"Добавь логотип Оптика Будущего"
"Сайт для Оптика Будущего в современном стиле"
```

**Автоматическое применение:**

Claude найдет логотип в `brands/оптика-будущего/assets/logos/` и применит его при генерации.

Когда guidelines будут заполнены, Claude также будет использовать фирменные цвета, шрифты и визуальный стиль.

## Задачи

1. Заполнить `guidelines/colors.txt` с фирменной палитрой
2. Заполнить `guidelines/typography.txt` с правилами шрифтов
3. Заполнить `guidelines/visual_style.md` с описанием визуального стиля
4. Добавить дополнительные варианты логотипа (белый, темный)
5. Добавить шаблоны презентаций (если есть)
6. Рассмотреть создание фирменного персонажа
